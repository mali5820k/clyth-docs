HTTPS / Server Runtime

The server module exposes one Clyth-facing server abstraction. Plain HTTP is used when TLS certificates are not configured; TLS can be layered through configuration without changing the routing model.

Router.get

void Router.get(string path, function<void(Request, Response)> callback)

Parameters

NameTypePurpose
pathstringRoute path.
callbackfunctionRoute callback invoked with borrowed request/response wrappers.

Returns

No return value.

Example

Router router = Router()

router.get("/", (Request request, Response response) => {
    response.send(200, "hello")
})

Router.post

void Router.post(string path, function<void(Request, Response)> callback)

Parameters

NameTypePurpose
pathstringRoute path.
callbackfunctionCallback that can read request data and send a response.

Returns

No return value.

Example

router.post("/api/message", (Request request, Response response) => {
    response.send(200, request.body())
})

Request.body

string Request.body()

Parameters

NameTypePurpose

Returns

Request body as a string value. The request wrapper is borrowed and should not be freed by user code.

Example

string body = request.body()
println(body)

Response.send

void Response.send(int32 status, string payload)

Parameters

NameTypePurpose
statusint32HTTP status code.
payloadstringResponse body.

Returns

No return value. Sends a response through the native runtime.

Example

response.send(200, "OK")

NameTypePurpose
