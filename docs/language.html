Clyth 0.5.0

Language Guide

Clyth keeps the alpha language surface explicit: declarations are readable, runtime ownership is visible at API boundaries, and higher-level convenience is added only when it does not hide systems behavior.

TypesVariables, arrays, strings, chars, and primitive values.FunctionsFirst-class function values and lambdas.StructsFields, methods, constructors, explicit this, and extension methods.GenericsGeneric collections and type parameters.StringsSSO direction, char arrays, and safe slices.

What 0.5.0 establishes

The 0.5.0 language subset is large enough to write meaningful runtime wrappers: structs can model Clyth-facing APIs, function values can cross router callbacks, and template literals can evaluate expressions.

include "https"

router.post("/api/message", (Request request, Response response) => {
    string body = request.body()
    response.send(200, `received ${body}`)
})
