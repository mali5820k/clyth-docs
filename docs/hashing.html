Hashing

The hash module uses rapidhash behind a Clyth API surface. Hashing is runtime functionality, not compiler magic.

hash_string

uint64 hash_string(string value)

Parameters

NameTypePurpose
valuestringInput text.

Returns

A uint64 hash value.

Example

include "hash"

uint64 id = hash_string("Clyth")
println(`${id}`)

hash_bytes

uint64 hash_bytes(char[] bytes)

Parameters

NameTypePurpose
byteschar[]Dynamic byte/character storage.

Returns

A uint64 hash value.

Example

char[] data = [67,108,121,116,104]
uint64 id = hash_bytes(data)
