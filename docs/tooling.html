Build Tooling

The current build scripts support compiler development, runtime archive builds, regression execution, ABI auditing, and cleanup of generated artifacts.

Common commands

Use the regression and ABI audit scripts before cutting a release tarball.

./build_compiler.sh --skip-dist
./run_regression_tests.sh
./audit_no_gnu_abi.sh
./scripts/clean_generated.sh
Clyth documentation is written for the 0.5.0 Runtime Ecosystem Foundation release.
