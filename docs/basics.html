Basics

This page introduces the core structure of a Clyth program before the detailed language and runtime references.

Program entry point

A native executable begins at main and returns an integer process status.

include "file-io"

int32 main() {
    println("Hello from Clyth")
    return 0
}

Variables

Declarations place the type before the variable name. Clyth uses explicit-width integers when the width matters.

int32 count = 24
bool complete = true
string release = "0.5.0 Alpha"

Structs and methods

Structs group fields and behavior. Methods use an explicit this receiver inside their body.

struct Counter {
    int32 value

    void increment() {
        this.value = this.value + 1
    }
}
