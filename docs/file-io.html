File I/O

The File I/O module exposes synchronous file operations through Clyth functions while routing implementation through the runtime provider. In 0.5.x this provider is libuv-backed.

read_file

string read_file(string path)

Parameters

NameTypePurpose
pathstringPath to the file to read.

Returns

A string containing file contents.

Example

include "file-io"

string text = read_file("config.json")
println(text)

write_file

void write_file(string path, string contents)

Parameters

NameTypePurpose
pathstringDestination path.
contentsstringText to write.

Returns

No return value. The call completes after the write operation finishes.

Example

include "file-io"

write_file("out.txt", "hello from Clyth")

file_exists

bool file_exists(string path)

Parameters

NameTypePurpose
pathstringFilesystem path to check.

Returns

true if the path exists, false otherwise.

Example

if file_exists("out.txt") {
    println("file exists")
}

delete_file

void delete_file(string path)

Parameters

NameTypePurpose
pathstringFile to remove.

Returns

No return value.

Example

if file_exists("out.txt") {
    delete_file("out.txt")
}
