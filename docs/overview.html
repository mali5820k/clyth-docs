Clyth 0.5.0

Overview

Clyth 0.5.0 is the Runtime Ecosystem Foundation release. It proves that Clyth code can compile to native executables, cross a stable ABI boundary, and drive useful native-backed runtime capabilities.

Current compiler path

The current bootstrap compiler lowers the supported language subset through LLVM IR generation. LLVM remains visible because it is the mature optimization and native-code backend used by this release, but Clyth's language model is not intended to be permanently coupled to LLVM's C++ APIs.

0.5.0 validation

  • 20+ regression samples covering structs, strings, functions, lambdas, modules, JSON, hashing, file I/O, and HTTPS routing.
  • A recursive ABI audit that rejects accidental GNU/glibc/libstdc++ runtime leakage.
  • A browser-facing demo where Clyth serves a page and accepts JSON POST payloads through a libuv-backed event loop.
Clyth documentation is written for the 0.5.0 Runtime Ecosystem Foundation release.
