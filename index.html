Version 0.5.0 Alpha

A working foundation for Clyth.

Clyth is an ahead-of-time systems programming language project that emits LLVM IR and produces native Linux programs through a musl-oriented toolchain. Version 0.5.0 Alpha is the released C++-hosted foundation; future alpha milestones will build the self-hosted compiler and evolve the language toward a stable 1.0 release.

Released foundation

0.5.0 Alpha includes a working compiler, runtime modules, a 24-test regression suite, and audited native release artifacts.

Self-hosting next

0.6.0 Alpha is intended to introduce a compiler written in Clyth, using the released compiler as its stage-zero foundation.

Stable destination

Milestones from 0.6.0 through 1.0 remain intentionally malleable. The destination is a self-hosted, maintainable systems language with MECC proper.
