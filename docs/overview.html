Clyth 0.5.0 Alpha

Version 0.5.0 Alpha is the first public alpha release of Clyth. It establishes a working compiler, runtime ecosystem, module model, regression suite, and native Linux release path.

Current release: 0.5.0 Alpha is released and validated by 24 passing regression programs.

What this release represents

The released compiler is implemented in C++ and translates Clyth source into LLVM IR. It is intentionally being stabilized rather than expanded indefinitely: its next responsibility is to compile the first compiler written in Clyth.

What comes next

The 0.6.0 alpha series begins the self-hosting transition. Later milestone numbers are planning markers rather than fixed promises; individual features may move between releases as the compiler is stress-tested. The long-term target remains a stable 1.0 release containing the intended language surface and MECC proper.
