JSON

The JSON module wraps yyjson through a small ABI layer. Parsed documents are owned resources and should expose free().

JSON.is_valid

bool JSON.is_valid(string text)

Parameters

NameTypePurpose
textstringJSON source text.

Returns

true if the input is syntactically valid JSON.

Example

include "json"

if JSON.is_valid("{\"ok\":true}") {
    println("valid")
}

JSON.parse

JsonDocument JSON.parse(string text)

Parameters

NameTypePurpose
textstringJSON source text.

Returns

An owned JSON document wrapper. Call free() when finished.

Example

JsonDocument doc = JSON.parse(request.body())
println(doc.to_string())
doc.free()
