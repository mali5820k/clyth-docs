Clyth 0.4.0-alpha

An LLVM-based systems programming language with a runtime-first architecture.

Clyth explores explicit control, predictable execution, and a smaller compiler boundary. The compiler understands the language; the runtime ecosystem provides capabilities.

Smaller Compiler

The compiler focuses on syntax, semantics, generic lowering, and LLVM IR generation.

Richer Runtime

Collections, hashing, file I/O, networking, and JSON belong in ordinary Clyth modules.

C ABI Boundary

Official native bindings target C and musl-friendly portability rather than C++ runtime dependencies.
