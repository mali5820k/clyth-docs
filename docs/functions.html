Functions, Callbacks, and Lambdas

Clyth functions support ordinary declarations, first-class function values, and lambda callbacks. The canonical function type syntax is function<Return(Args...)>.

Ordinary functions

Functions declare their return type and parameter list explicitly.

int32 add(int32 left, int32 right) {
    return left + right
}

First-class functions

Function values can be passed to other functions using function<Return(Args...)>.

int32 run(function<int32(int32)> callback) {
    return callback(5)
}

int32 square(int32 value) {
    return value * value
}

Lambda callbacks

When a lambda omits an explicit return type, no return statements infer void. Return statements with values infer the returned type if all return paths agree.

router.post("/", (Request request, Response response) => {
    response.send(200, request.body())
})

function<int32(int32)> doubler = (value) => {
    return value * 2
}
