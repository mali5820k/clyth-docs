Release status

This page distinguishes shipped 0.5.0 behavior from future design work. A feature described as planned or under research is not part of the current language contract.

Released

0.5.0 Alpha

  • C++-hosted ahead-of-time compiler
  • LLVM IR generation
  • 24 of 24 release regression programs passing
  • musl-oriented native release path
  • runtime collections, file I/O, JSON, hashing, and HTTPS/router modules
  • structs, methods, generics, lambdas, first-class functions, arrays, formatted literals, and native ABI integration
  • deterministic primitive default initialization; uninitialized strings begin as null runtime values

Current focus

0.6.0 Alpha

  • compiler implementation written in Clyth
  • compact, readable lexer and Pratt parser
  • Clyth-owned AST and IR structures
  • textual LLVM IR emission
  • stage-zero, stage-one, and self-rebuild validation
  • smaller toolchain surface without LLVM C++ development libraries or libc++ in the self-hosted compiler

Design and research

Beyond the released contract

  • richer value-oriented string semantics while preserving deterministic initialization
  • automatic temporary lowering and mutable parameter aliasing
  • nullable flow analysis
  • ownership inference and generated cleanup
  • MECC estate analysis and collapse
  • string interning and other storage optimizations

Milestones are malleable

Version labels between 0.6.0 and 1.0 describe the current development sequence, not immutable feature deadlines. Stress-testing the language by writing its compiler in Clyth may reveal changes that move, split, simplify, or replace planned features. The stable destination is more important than preserving an early milestone assignment.
