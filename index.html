Clyth 0.5.0 · Runtime Ecosystem Foundation

Systems programming with a clear runtime boundary.

Clyth is an LLVM-backed ahead-of-time systems language project focused on native Linux artifacts, explicit language semantics, and a runtime model where Clyth wrappers call through small auditable ABI layers into native providers such as libuv, yyjson, and rapidhash.

Compiler-owned semantics

Clyth owns syntax, typing, semantic analysis, and lowering. LLVM is used as a mature backend, not as the identity of the language.

Native capability through wrappers

Runtime features are exposed as Clyth APIs while implementation details stay below a stable ABI boundary.

Auditable artifacts

The 0.5.0 release candidate validates native artifacts through musl/libc++ builds and recursive GNU/glibc/libstdc++ audit checks.
