Roadmap to 1.0

The roadmap communicates direction rather than immutable feature deadlines. Alpha milestones may change as the self-hosted compiler exposes new constraints and opportunities.

Completed — 0.5.0 Alpha

  • released C++-hosted compiler and native build pipeline
  • 24 passing regression programs
  • runtime modules and stable C ABI layering
  • collections, JSON, hashing, file I/O, and HTTPS/router support
  • documentation, release packaging, and GNU ABI audit tooling

In development — 0.6.0 Alpha

  • create the compiler written in Clyth
  • introduce a readable lexer and Pratt parser
  • represent syntax, semantic information, and IR using Clyth types
  • emit textual LLVM IR without linking the compiler against LLVM C++ libraries
  • build against a native musl toolchain and remove libc++ from the self-hosted compiler path
  • validate stage-one and stage-two rebuilds alongside the language regression suite

Planned alpha milestones — 0.7.0 and later

Potential work includes richer string semantics—including replacing the 0.5.0 null-string default with a value-oriented representation while preserving deterministic initialization—compiler-managed temporaries, improved diagnostics, nullable flow analysis, optimizer passes, module/distribution refinement, additional native targets, and ownership analysis. Their exact version assignment may change.

Vision — 1.0.0 Stable

The stable target is a self-hosted systems language with a maintainable compiler, predictable native behavior, a coherent runtime/module ecosystem, stable language and ABI contracts, and MECC proper providing estate-oriented ownership analysis and cleanup.
