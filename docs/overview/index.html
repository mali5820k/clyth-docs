Overview

Clyth 0.5.0

The Runtime Ecosystem Foundation release proves Clyth can compile native programs, cross a stable ABI boundary, and drive useful runtime capabilities through ergonomic Clyth APIs.

Architecture in one picture

Clyth keeps the public API in Clyth and hides external libraries behind a small ABI. This lets modules scale without turning the compiler into a framework.

Clyth application
    ↓
Clyth wrapper module
    ↓
Stable C ABI boundary
    ↓
Native provider
    ↓
libuv / yyjson / rapidhash / OS

Web backend proof

The 0.5.0 demo validates browser request handling, libuv event-loop lifetime, llhttp parsing, Clyth route callbacks, and JSON response flow.

include "https"

int32 main() {
    Router router = Router()

    router.post("/api/message", (Request request, Response response) => {
        response.send(200, request.body())
    })

    Server server = Server(ServerConfiguration(9090, router))
    server.start()
    return 0
}
