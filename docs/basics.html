Basics

The basics page is intentionally short: enough to orient new readers before they move into the language and runtime guides.

Main and arguments

A Clyth executable starts at main. Command-line arguments are available through string[] args.

int32 main(string[] args) {
    println(`${args.length}`)
    return 0
}

Structs and methods

Structs define data. Methods define behavior. The behavior may live inside the struct declaration or as an extension method.

struct Counter {
    int32 value
}

void Counter.increment() {
    this.value = this.value + 1
}
