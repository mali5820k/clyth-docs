Getting Started

This section will describe how to build Clyth, run the regression suite, and compile a first program.

./build_compiler.sh --skip-dist
./sample-clyth-programs/run_language_samples.sh

Clyth currently targets a source-built compiler workflow. The 0.5.0 documentation effort will introduce cleaner project-level commands and module workflows.
